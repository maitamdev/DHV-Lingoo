# Changelog

All notable changes to DHV-Lingoo will be documented here.

## [1.0.0] - 2024-03-03

### Added
- Landing page with Hero, Features, FAQ, Testimonials, Pricing
- User authentication (login, register)
- Dashboard with real-time stats, charts, leaderboard
- Course management with lessons and vocabulary
- Dictionary with Free Dictionary API + AI Vietnamese translation
- AI Chatbot (Lingoo Fox) powered by Groq
- Push notification study reminders
- User settings and profile management
- Learning roadmap visualization
- Subscription management
- Flutter mobile app with all core features

### Technical
- Next.js 15 with App Router
- Supabase for database and auth
- Groq LLM for AI features
- Recharts for data visualization
- Flutter 3.27 for mobile app
- GoRouter for Flutter navigation
- fl_chart for Flutter charts
