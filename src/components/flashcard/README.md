# Flashcard Components

## Components
- MysteryBag - Unopened card bag with tap-to-reveal
- FlashcardItem - Revealed card with 3D flip
- ParticleEffect - Burst particles on bag open
- RarityBadge - Rarity label
- DailyProgress - Progress bar
- ConfettiEffect - Celebration on completion
- CompletionCard - Summary with XP
- DailyHeader - Date and title
- FlashcardStats - Stats summary
- NextDayCountdown - Timer to midnight
- CardCollection - Gallery of revealed cards
- FlashcardSkeleton - Loading placeholder
