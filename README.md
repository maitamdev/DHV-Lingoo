# DHV-Lingoo

Nen tang hoc tieng Anh hien dai voi AI, flashcards va theo doi tien trinh.

## Tech Stack
- **Web**: Next.js 15, Supabase, Groq AI, Recharts
- **Mobile**: Flutter 3.27, Supabase SDK, fl_chart, GoRouter

## Getting Started

### Web
npm install && npm run dev

### Flutter
cd dhv_lingoo_app && flutter pub get && flutter run

## Features
- Interactive lessons with TTS
- Real-time XP and progress tracking
- Leaderboard and achievements
- EN-VI Dictionary
- AI Chatbot (Lingoo Fox)
- Push notification reminders
- Flutter mobile app
