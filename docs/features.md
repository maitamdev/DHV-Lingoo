# Features

## Core Features
1. Dashboard - Learning progress overview
2. Courses - Structured English lessons
3. Flashcards - Daily vocabulary with rarity system
4. Practice - Vocab quiz, fill-blank, listening
5. Review - Spaced repetition vocabulary review
6. Dictionary - EN-VI dictionary and translation
7. Chat AI - Conversational AI tutor (Fox mascot)

## Gamification
1. XP System - Earn XP from all activities
2. Streak System - Daily login streaks
3. Achievements - 50+ unlockable badges
4. Leaderboard - Compete with other learners
5. Level System - A1 to C2 progression

## Tools
1. Vocabulary Notebook - Personal word lists
2. Study Planner - Daily task scheduling
3. Profile - User statistics and charts
4. Settings - Profile and preferences
