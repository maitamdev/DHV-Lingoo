# API Reference

## Vocabulary
- GET /api/vocabulary?search=word&limit=50&offset=0

## Leaderboard
- GET /api/leaderboard?sort=xp&limit=50

## Profile
- POST /api/profile/xp - Award XP
- POST /api/profile/streak - Update streak

## Progress
- GET /api/progress - User progress summary

## Achievements
- POST /api/achievements/check - Check and unlock

## Review
- POST /api/review/save - Save review session

## Dictionary
- POST /api/translate - Translate or lookup word
